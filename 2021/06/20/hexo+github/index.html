github + hexo

本人对于这方面基本都是零基础,但是我就是想试一试,一个零基础得,毫无计算机相关经验得人,能否成功用github+hexo搭建成一个自己得博客.

由于笔者之前就有过类似得想法,并且尝试过,但止步于安装完node+git+hexo,往后面得步骤总是莫名奇妙的错,让我放弃,时隔一年,突然记起来电脑还因为当年得这件事情,下载好了node.js hexo等等一系列渣七乱八得东西,决定重新试一试,看看能不能搭建成功.

声明:本文一切全是来源于网络,自己只是把自己得按照网络上的步骤一步步详细记录下来.
